## Purpose
This script automatically sets the `marker-color` for shops by the different network speed.

Color | HEX | Speed
------ | ------ | -----
RED | #C24740 | 0~5M
YELLOW | #F3AE1A | 5~10M
GREEN | #50C240 | 10~M

## Prequisite
[Node.js](https://nodejs.org/)

## How to run

`node tool/build_marker.js`
